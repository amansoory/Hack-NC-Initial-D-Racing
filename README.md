# Initial-D Racing Game 🏎️💨

A 3D arcade-style racing game inspired by **Initial D**, built in **Unity** during **HackNC 2024**. The game focuses on fast-paced driving mechanics, drifting, and responsive controls in a stylized mountain racing environment.

## Overview
This project was developed as a hackathon submission and explores the fundamentals of **Unity game development**, including physics-based vehicle movement, camera systems, and game asset integration.

Players control a racing car navigating a mountain track, emphasizing **speed, control, and drifting mechanics** inspired by classic street racing games.

## Features
- 🚗 Physics-based vehicle movement and steering
- 🎮 Responsive player controls for acceleration, braking, and turning
- 📷 Dynamic camera system that follows and locks onto the vehicle
- 🛞 Wheel and suspension setup using Unity physics components
- 🎨 Integration of 3D assets and environment elements
- 🏁 Arcade-style racing gameplay inspired by *Initial D*

## Tech Stack
- **Game Engine:** Unity  
- **Language:** C#  
- **Tools:** Unity Editor, Visual Studio / VS Code  

## Learning Focus
This project helped explore:

- Unity physics systems  
- Camera control and object tracking  
- Asset management and game scene composition  
- Debugging Unity scripts and component interactions
